# AWS S3 Static Website Hosting

## 📌 Project Overview
This project demonstrates how to host a static website using AWS S3.

## 🚀 Services Used
- AWS S3
- IAM

## ⚙️ Steps
1. Create S3 bucket
2. Upload website files
3. Enable static hosting
4. Set bucket policy

## 📂 Features
- Highly available
- Cost-effective
- Easy deployment

## 🌐 Output
Static website hosted using AWS S3.
